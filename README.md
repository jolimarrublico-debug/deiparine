## How to Use

1. Open `index.html` in your web browser
2. Navigate through slides using:
   - **Arrow buttons** on the sides
   - **Dot indicators** at the bottom
   - **Keyboard arrows** (left/right)
   - **Auto-play** - slides advance automatically every 3 seconds

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and animations
- `script.js` - Slide show functionality and controls

## Customization

To customize the slide show:

1. **Add/Remove Slides**: Edit the HTML file to add or remove `.slide` divs
2. **Change Auto-play Speed**: Modify the interval time in `script.js` (default: 3000ms)
3. **Update Styling**: Customize colors and animations in `styles.css`

## Browser Compatibility

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
